## 设计规格

**配色方案：**
- 背景色：#FAF9F5（米白）
- 主色：#FF9882（珊瑚橙）、#FF6B6B（红橙渐变）
- 卡片白：#FFFFFF
- 文字：#333333、#666666、#999999
- 状态色：amber/orange/rose/indigo 对应各餐次

**餐次配色：**
- 早餐：amber-50 bg + amber-500 text
- 午餐：orange-50 bg + orange-500 text
- 下午茶：rose-50 bg + rose-400 text
- 晚餐：indigo-50 bg + indigo-400 text

**圆角规范：**
- 大卡片：24rpx
- 中卡片：20rpx
- 小元素：12rpx

## 底部导航结构

```vue
<view class="bottom-nav">
  <view class="nav-item">首页</view>
  <view class="ai-btn-center">
    <!-- 凸起的AI按钮 -->
  </view>
  <view class="nav-item">我的</view>
</view>
```

**样式要点：**
- `position: fixed; bottom: 0`
- 中间按钮 `transform: translateY(-24rpx)`
- `border-radius: 32rpx` 顶部圆角
- `backdrop-filter: blur` 毛玻璃效果
