# 宝宝辅食小程序页面重构设计

## 概述

对小程序首页、食谱详情、档案页、历史页进行整体重构，采用温暖亲子风格，优化信息架构和操作动线。

## 设计风格

- **主色调**：#FF9B5E（暖橙）
- **背景色**：#FFF8F3（米白）
- **卡片色**：白色 + 微阴影
- **文字色**：#333（深灰）、#666（次级灰）
- **餐次色块**：早餐-浅橙、午餐-浅绿、下午茶-浅紫、晚餐-浅蓝
- **风格**：圆润卡片、柔和配色、温暖亲子感

---

## 一、首页（index.vue）重构

### 顶部区域
- 左侧：宝宝头像占位（圆形）+ 昵称 + 月龄
- 右侧：今日日期
- 问候语：「今天想给宝宝做什么？」

### 有食谱时
- 主卡片显示今日四餐缩略信息
- 每餐一行：餐次色块 + 名称 + 食材数量
- 底部「查看今日食谱」主按钮
- 右下角「重新生成」小字入口

### 无食谱时
- 居中大按钮「生成今日食谱」
- 下方小提示「录入宝宝信息，获取更精准的食谱」

### 底部快捷入口
- 横排两个入口：档案管理 | 历史记录
- 各带图标 + 文字标签

---

## 二、食谱详情页（recipe.vue）重构

### 整体布局
- 顶部固定 Header：日期 + 「今日食谱」
- 四餐分块展示（纵向滚动）

### 每餐卡片结构
```
┌─────────────────────────────────┐
│ [色块] 早餐            20分钟  │
├─────────────────────────────────┤
│ 南瓜小米粥                      │
│                                 │
│ [食材标签] [食材标签] [食材标签] │
│                                 │
│ 点击展开查看详情              ▼  │
└─────────────────────────────────┘
```

### 展开后详情
- 步骤以数字序号纵向排列
- 每步独立一行，带序号圆圈
- 步骤间距24rpx

### 餐次色块
- 早餐：浅橙色
- 午餐：浅绿色
- 下午茶：浅紫色
- 晚餐：浅蓝色

### 底部操作栏
- 「采纳这份食谱」主按钮
- 「重新生成」次按钮
- 「记录成果」文字链接

---

## 三、档案页（profile.vue）重构

### 页面结构
- 顶部：页面标题「宝宝档案」+ 编辑/保存按钮
- 信息分组展示

### 分组一：基本信息
- 昵称（输入框）
- 出生日期（日期选择器）
- 性别（单选：男/女）

### 分组二：健康信息
- 过敏原（多选标签：鸡蛋、牛奶、坚果、鱼类、虾蟹、大豆、小麦）
- 口味偏好（多选标签：甜的、酸的、软糯、细腻）
- 不喜欢（多选标签）

### 分组三：地区信息
- 城市（地区选择器）

### 视觉风格
- 分组之间用细线分隔
- 每组标题字号24rpx，颜色#999
- 输入框/选择器用浅灰背景+圆角

---

## 四、历史页（history.vue）重构

### 顶部
- 页面标题「历史记录」
- 筛选 Tab：全部 / 本周 / 本月

### 列表形式
- 按日期分组，每组：
  - 日期标签（今天/昨天/03月15日）
  - 卡片列表

### 每条记录卡片
```
┌─────────────────────────────────┐
│ 03月15日 · 午餐                 │
├─────────────────────────────────┤
│ 南瓜小米粥                      │
│ 鸡蛋、南瓜、小米               │
│                                 │
│ [已采纳]            [查看] [复用]│
└─────────────────────────────────┘
```

### 状态标签
- 待反馈：橙色
- 已采纳：绿色
- 未采纳：灰色
