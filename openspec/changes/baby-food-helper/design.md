# Design: baby-food-helper

## Context

宝宝辅食助手是一款帮助0-3岁宝宝家长解决每日辅食选择困难的微信小程序。通过AI技术，根据宝宝月龄、体质、口味，一键生成科学的每日辅食安排。

### 当前状态
- 全新项目，无历史包袱

## Goals / Non-Goals

**Goals:**
- 实现一键生成每日辅食的核心功能
- 支持多宝宝档案管理
- 试用生成无需注册即可体验
- 简洁高效的产品体验

**Non-Goals:**
- 推送服务（用户主动打开）
- 电商功能
- 社区分享
- 复杂的营养分析

## Decisions

### 1. 使用微信云开发
免费额度够MVP阶段使用，部署简单

### 2. 试用生成月龄不存储
只在当次生成中使用，不持久化，减少复杂度

### 3. AI去重策略
传入7天历史摘要，让AI智能决策是否换口味

### 4. 设计风格
- 简洁风格，去AI化（少用emoji）
- 功能优先，不花哨

### 5. 一日多餐展示
单次生成展示全天辅食安排（早餐+午餐+下午茶+晚餐）

## Risks / Trade-offs

### Risks
- AI生成可能不符合口味，需要用户反馈优化
- MiniMax API 服务稳定性

### Trade-offs
- 简化推送功能，依赖用户主动打开
- 不做复杂营养分析，保持简洁
